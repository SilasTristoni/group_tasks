🧪 Como Testar cada Exercício
Ao abrir o menu, escolha o número correspondente para avaliar cada lógica:

[1] Sistema de Biblioteca: Apenas observe a saída no terminal. O código demonstrará a criação de objetos e a herança entre as classes.

[2] Consulta de Clima: Quando o sistema pedir, digite as coordenadas sugeridas na tela (ex: Lat -26.30 e Lon -48.84) para testar o consumo da API externa com Future.

[3] e [4] Chat via Sockets (TCP):

Abra dois terminais diferentes na pasta do projeto.

No primeiro terminal, inicie o sistema e escolha a Opção 3 (Servidor).

No segundo terminal, inicie o sistema e escolha a Opção 4 (Cliente).

Troque mensagens livremente entre as janelas.

Digite a palavra sair e pressione ENTER para finalizar o chat sem travar o menu.

[5] Refatoração Null Safety (Ex. 4): O sistema pedirá um ID de funcionário. Digite 1 ou 2 para ver o comportamento com dados preenchidos. Digite 3 (ou aperte ENTER sem digitar nada) para testar a proteção do código contra valores null.
